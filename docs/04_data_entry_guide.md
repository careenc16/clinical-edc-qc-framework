# DATA ENTRY GUIDE FOR SITE COORDINATORS
## Study XYZ-2026 — Phase II Diabetes Trial

### 1. General Rules
- All dates must be entered as **YYYY-MM-DD** (e.g., 2026-03-20)
- All required fields (marked with *) must be completed before saving
- Do NOT leave required fields blank — the system will generate a query
- When in doubt, check the source document first

### 2. Subject ID Format
Format: **XYZ-2026-SSS-PPPP**
- XYZ-2026 = study identifier (fixed)
- SSS = your 3-digit site number (e.g., 001, 002)
- PPPP = 4-digit patient number (sequential at your site)
- Example: XYZ-2026-003-0015

### 3. Demographics (DM)
- **Informed Consent Date**: Must be on or before the first dose date
- **Age**: Will be auto-checked against date of birth — if they don't
  match, you'll receive a query
- **Sex**: Select M (Male) or F (Female) only
- **Treatment Arm**: Auto-assigned by IXRS — do not modify

### 4. Adverse Events (AE)
- Report ALL adverse events, regardless of relationship to study drug
- **Start Date**: Must be on or after the informed consent date
- **End Date**: Required if outcome is "Recovered/Resolved"
- **Severity vs Serious**: These are DIFFERENT concepts
  - Severity = how bad (Mild/Moderate/Severe)
  - Serious = meets ICH criteria (hospitalization, death, disability)
  - A MILD AE can be SERIOUS (e.g., mild allergic reaction requiring
    hospitalization for observation)
  - If you mark SEVERE, the system will ask you to confirm if it's also SERIOUS

### 5. Vital Signs (VS)
- Acceptable ranges (values outside these will trigger a query):
  - Systolic BP: 70–250 mmHg
  - Diastolic BP: 40–150 mmHg
  - Heart Rate: 40–180 bpm
  - Temperature: 35.0–40.0 °C
- Systolic must ALWAYS be higher than Diastolic
- If a value is truly outside range, verify it and respond to the query
  with "Confirmed per source document"

### 6. Responding to Queries
When you receive a data query:
1. Review the query text carefully
2. Check the source document (patient chart)
3. If the data is wrong → correct it in the EDC and note what you changed
4. If the data is correct → respond with "Confirmed as entered per source"
5. Respond within 5 business days

### 7. Common Mistakes to Avoid
| Mistake | How to Prevent |
|---------|---------------|
| Wrong date format | Always use YYYY-MM-DD |
| Missing AE end date | Enter end date when AE resolves |
| Entering free text in coded fields | Select from dropdown only |
| First dose before consent | Double-check consent date |
| Systolic/diastolic swapped | Systolic is always the higher number |
